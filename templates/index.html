Music Genre Classification
A Web Application by: Yash Chauhan

GitHub: ML Model Development GitHub: Model Deployment

About:
Want to find out which music genre gives you the kicks, upload some of your favourite songs and this tool will use the power of machine learning(Really Smart Mathematics) and return the name of the most probable genre for that song.

Under the hood:
You can specify which Machine learning model you want to use, The application supports XGBoost, CatBoost and Random Forest Algorithms. Another option is to use the ensembled model I have created that has greater accuracy than any of the former ones. If you want to know about the Development of the ensembled model you can check the given github repository other than that try the models and go nuts.
